Mark Allman wrote:
> I think there IS a difference. The initial window burst happens
> once per connection (OK, maybe more if it goes idle). If you use an
> ACK interval of 4 segments the burst will happen much more often.
> Not saying I think the burstiness that would be caused by an ACK
> interval of 4 segments is acceptable or not, but I think it is
> sufficiently different from the larger initial window proposal.

I'm starting to generalize here - away from the specific of a
four-segment ACK and more towards generic ACK avoidance.

I presume it would be bad if that added burstiness led to increased
packet drops, but if no addional packet drops resulted (say in an
intranet) there does seem to be a benefit as evidenced by the original
poster's observed 8% gain.

If we assume that added packet drops resulted, what are the chances
(under RED I presume) that the packet drops would be suffered by the
non-bursty connections as opposed to ones that were not bursty. In other
words, how likely is it that someone else feels the pain instead?

Now, I suspect that the original person's test was a long-lived
connection like a netperf TCP_STREAM test, so the issue of initial slow
start was moot, and the real world (tm) would not allow that to be moot.

In that case I suppose that a reciever wishing to avoid ACK's would have
to have some smarts to know that slow-start was happening and ACK
accordingly. If a burstiness resulted in packet losses for that
connection, the receiver would see out of order packets, start sending
immediate ACKs and (presumeably) start using an ACK policy sympatico
with fast recovery/retransmit.

Shouldn't the sender then arrive at a cwnd that was sufficient to allow
his packets through in those bursts, and also have that sender (assuming
it is only incrementing by one MSS per ACK regardless of the number of
segments ACKed) be more conservative on its probing for additional cwnd?

I wonder if there is a theoretical sweet-spot somewhere for ACKs as a
fraction of the perceived cwnd. As in an ACK for every 8% of cwnd, 25%,
50%.
