A graphical TCP connection analysis tool: "tcpillust" is now available at:

        ftp://ftp.csl.sony.co.jp/CSL/nishida/tcpillust.tar.gz

Tcpillust takes tcpdump file(s) specified at the command line and draw
pictures like figures in the ``TCP/IP Illustrated'' series.

If you're interested, please check my web site:
        http://www.csl.sony.co.jp/person/nishida/tcpillust.html
You can see sample screen images of tcpillust.

Currently, I tested only on freebsd-3.4 with tcl8.2/tk8.2.
If you have any problem for building tcpillust, please let me know.
     
Feedbacks, comments, suggestions and bug-reports are welcome.

P.S.
if you're looking for a presentation software on UNIX, please check
